The not-null assertion operator !! makes sure the given nullable variable in fact is not null, and throws a NullPointerException if it is. It actually is the only way to have such an exception thrown in Kotlin. It is rarely required (mostly in legacy code) and should be used sparingly. Example: 
val s : String? = null
val l = s!!.length // will throw NullPointerException
